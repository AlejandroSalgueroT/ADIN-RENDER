Should Have - Advice or Criticism

¿Qué es "Should Have"?

Should have se usa para dar consejos o expresar críticas sobre el pasado. Indica lo que alguien debió haber hecho.

  • Ejemplo: You should have studied.

Estructura de 'Should Have'

  • Afirmativa: Sujeto + should have + participio pasado
    Ejemplo: She should have arrived earlier.
  • Negativa: Sujeto + shouldn't have + participio pasado
    Ejemplo: They shouldn't have left so soon.
  • Interrogativa: Should + sujeto + have + participio pasado?
    Ejemplo: Should you have told her?

Esta estructura se usa para expresar consejos, arrepentimientos o críticas sobre acciones que ocurrieron (o no ocurrieron) en el pasado.
