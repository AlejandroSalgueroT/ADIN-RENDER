Should in Simple Tenses

¿Qué es "Should"?

Should es un verbo modal usado para dar consejos, recomendaciones o expresar lo que es correcto hacer.

Estructura (Simple Tenses)

  • Afirmativa: Subject + should + base verb → You should sleep.
  • Negativa: Subject + should not (shouldn't) + base verb → You shouldn't eat that.
  • Interrogativa: Should + subject + base verb? → Should we leave?
