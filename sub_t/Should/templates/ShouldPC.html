Should Have Been - Perfect Continuous

¿Qué es "Should Have Been"?

Should have been se usa para expresar una acción continua que debió estar ocurriendo en el pasado, pero no fue así. Se emplea para dar consejos o críticas sobre acciones prolongadas que no se realizaron.

Estructura

  • Forma: Subject + should have been + verb-ing → You should have been studying.
